# ArangoDB GO Driver Roadmap.

## Version 1.0 
- Expected May 2017 (aligned with ArangoDB 3.2 release)
- Stable API 
- Supports:
    - ArangoDB 3.1 and up.
    - Single server & cluster.
    - Authentication & secure connections.
- Connection using:
    - HTTP(s) + JSON
    - HTTP(s) + Velocypack 
    - Velocystream
